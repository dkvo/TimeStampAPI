API Project: Time Stamp API

    User Direction:

  1. The API end point is GET https://timestmp-api.herokuapp.com/timestamp/:date?.
  2. This API take user provided date string in date(YYYY-MM-DD, YYYY,MM,DD etc.) format and unix epoch(integer) format and return both unix epoch and UTC formatted timestamps.
  3. If error occur an object with error message "invalid date" will be returned.
  4. If no date was provided, the api will retrieve the current date timestamp.

Usage:

https://timestmp-api.herokuapp.com/timestamp/2018-01-23
https://timestmp-api.herokuapp.com/timestamp/1535347646303

Example Output:

{"unix":1535347646303,"utc":"Mon, 27 Aug 2018 05:27:26 GMT"}
